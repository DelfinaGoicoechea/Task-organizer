# Task Organizer

A simple task management application built with React and TypeScript that allows users to create, manage, and organize daily tasks efficiently.

## Description

Task Organizer is a To-Do application designed to practice modern frontend development concepts such as component architecture, state management and clean UI interactions.

Users can:

- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Receive validation feedback when inputs are invalid

## Getting Started

- Clone the repository
```sh
git clone git@github.com:DelfinaGoicoechea/Task-organizer.git
```
- Install dependencies
```sh
npm install
```
- Run the development server
```sh
npm run dev
```
